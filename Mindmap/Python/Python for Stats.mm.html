Python for Stats

  • numpy
    • r = numpy.arange(start, end, diff) 创建array
    • + - * /
    • numper.power(r,2) 平方
    • r>1 - 生成真值array
      • r[r>1] - 使用真值array过滤原array
    • 矩阵
      • numpy.dot(r,r.T)
  • pandas
    • pandas.Series
      • 定义：单列数据
      • 生成
        • x = Series([data_list]), 默认index
        • x = Series([data_list],index=[index_list]) 指定index
      • 查看
        • x[index], 0~len(x)
        • 支持切片
      • 插入
        • x.append[new_Series]
        • Series无法插入值，需要append序列
      • 查找判断
        • value in x.values
      • 删除
        • x.drop(index), 根据index
        • x.drop(x.index[loc])，根据位置
        • x[condition] 根据值删除
    • pandas.DataFrame
      • 定义：多行多列数据
      • 生成
        • df =DataFrame({dict})
        • key为列名，value为[list]
        • df = DataFrame({dict},index=[index_list])
      • 查看
        • df[key], df[[keys]] 列查看
        • df[切片] 行查看
        • df.loc[index],df.loc[[indexs]]行索引查看
        • df.iloc[切片],行列号访问
        • df.at[行，列]精准定位
      • 修改名称
        • df.columns = 修改列名 
        • df.index = 修改index
      • 插入
        • df.loc[len(df)] =[行，列] 添加行
        • df[列名] = [data] 添加列
      • 删除
        • df.drop(index,axis)
          • axis = 0 行
          • axis = 1 列
      • 查找
        • df.apply(min) - 各列最小值
        • df.apply(min,axis = 1) - 各行最小值
        • df.apply(lambda x: numpy.all(x>0), axis =1) - 判断所有列是否大于零
